#'stroids
A simple Asteroids clone for the Arduboy.

## Sidenote
I'm unsure if this runs on a production Arduboy. Works flawlessly on a Devkit unit.

## Controls 

**Gameplay**

| **Button** | **Action** |
|------------|------------|
| UP | Move |
| LEFT/RIGHT | Aim |
| DOWN | Warp |
| B (Left) | Pause & Unpause |
| A (Right) | Shoot |

**Menu**

| **Button** | **Action** |
|------------|------------|
| B (Left) | Start |
| A (Right) | Toggle Debug |
